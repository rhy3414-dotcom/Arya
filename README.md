# Arya Jha — Portfolio & Receipt

A beautifully designed interactive portfolio styled as an official receipt. Built with vanilla HTML, CSS, and JavaScript.

## 🚀 Quick Deployment to GitHub Pages

Your portfolio is **production-ready**! Just enable GitHub Pages:

### Step 1: Enable GitHub Pages (2 minutes)
1. Go to: **[Settings → Pages](https://github.com/rhy3414-dtcom/Arya/settings/pages)**
2. Under **Source**, select: **"Deploy from a branch"**
3. Choose branch: **`main`**
4. Click **Save**

### Step 2: Wait & Access
- Wait 2-3 minutes for deployment
- Your portfolio will be live at: **https://rhy3414-dtcom.github.io/Arya/**
